浮动
float：left right none;
当你加上浮动属性之后 就相当于 他是飘起来 飘在了其他的模块上面
之后其他的模块会 顶替他的位置 而且 都采用浮动之后 采用见缝插针的原则

清浮动:
1.对浮动的块 写固定的高度；
2.清浮动 clear ： none left right； 加在后面的盒子上面
3.当前浮动元素后面补一个盒子，不设置宽高，clear：both；在不浮动的模块上面添加
4.overflow：hidden，让浮动元素计算高度； 这个也是在浮动的元素上写
盒子模型
盒子模型就相当于 有一个快递盒子 盒子里面有一个商品
内边距padding:相当于时盒子和商品之间的间隙
取值： 1个值 四个方向一样；
2个值：上下 左右；
3个值：上 左右 下；
4个值；上 右 下 左；
padding特点：
1.背景色延伸至padding;
2.可以设置单一方向：padding-top，bottom ，left right
边框border：10px dotted red;相当于快递盒子
样式：solid，double，dash，dotted；
border-width：取值参考和padding一样
border-color：取值与padding一样 但是采取的是颜色
border-style：取值和padding一样 但是是样式的值
背景色也蔓延到边框
外边距margin：相当于盒子与盒子之间的间隙，可以设置成负数
margin取值方式与padding完全一致
横向居中方案 margin：0 auto；


特征问题：
1.margin兄弟问题：上下 取最大值 左右合并 2.父子添加margin：
（1）子margin-top = 父padding-top，注意高度计算； （2）给父盒子设置边框
（3）加浮动
（4）overflow：hidden

块元素：
1.默认情况下，块状元素会占据一行；多个块级元素会自上而下排列；
2.块状元素可以定义自己的宽度和高度；
3.块元素一般作为其他元素的容器
例如：div p ul li h1-h6
内联元素：
1.横着排
2.内联元素没有自己的形状，不能定义它的宽和高；
3.不能放块级元素
4.行内元素只能设置左右边距 没法设置上下边距
例如：a b em等
行内块元素：
同时具有行内内联元素和块状元素的特点
既可以调整大小 又可以和别人共享一个空间

定位
position：
static 默认值 默认
absolute 绝对定位 脱离文档流 当没有父元素或者父元素没有定位，参照物是浏览器第一屏
有父元素且父元素有定位的时候 参照物为父元素
relative 相对定位 不脱离 自己的初始位置
fixed 固定位置 脱离 浏览器的当前窗口
sticky 粘性定位 可以做吸顶效果
当设置定位之后 可以用top left right bottom来确定自己的位置
z-index可以利用层级来确定那个位置在上面
