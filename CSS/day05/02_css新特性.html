设置：display：flex
影响：
1.子元素默认横向排列
2.行内元素，变成了块级元素
3.只有一个元素时，margin：auto自动居中
有主轴还有侧轴：比如主轴是横向 那么侧轴就是纵向 这时候盒子排列方向是横向排列； 设置主轴：flex-direction：column；
column 从上到下排列；
row；从左到右排列；
column-reverse：从下往上排列；
row-reverse：从右往左排列；
调整主轴对齐方式

值：flex-start；以row方式为例 向左对齐 flex-end；以row为例 向右对齐 center：中间对齐 space-between 两端对齐
space-around 距离环绕 每两个元素之间距离相等
