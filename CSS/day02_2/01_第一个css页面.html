文本属性：
font：倾斜 加粗 字体大小/行距 字体;(倾斜和加粗可以省略 但是剩下的三个一定不能省略)
text-transform:uppercase,lowercase,capitalize; 大写 小写 首字母大写
letter-spacing:控制文字与文字之间的间距
text-decoration:none,underline,overline,line-through;
text-indext:首行缩进 一般用em单位来进行缩进
line-height:行高; line-height=height 实现单行文本居中 仅适用于单行文本
font-size:字体大小；单位px
font-family：多个字体中间用逗号链接 先解析第一个字体 依次往后推
color：三种方式 第一种 red 第二种 #fff 第三种 rgb(0,25,255){0-255}
font-weight:数字：100-500不加粗，500-900加粗;文字 normal常规，bold加粗，bolder更粗
font-style:italic 斜体字 oblique 倾斜的文字 normal常规
text-align:left 水平靠左 right 水平靠右 center水平居中 justify水平两端对齐 只对多行起作用
